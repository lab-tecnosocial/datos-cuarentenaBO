datos-cuarentenaBO

Documentación de la API estática para el proyecto "datos-cuarentenaBO". El repositorio se encuentra en GitHub

La URL base es:
https://lab-tecnosocial.github.io/datos-cuarentenaBO

gamlp

Datos del Gobierno Autonomo Municipal de La Paz

Up 
get /v1/gamlp/Abasto.json
Centros de abasto en tiempos de cuarentena (v1GamlpAbastoJsonGet)
Up 
get /v1/gamlp/EntidadesFinancieras.json
Entidades financieras (v1GamlpEntidadesFinancierasJsonGet)
Up 
get /v1/gamlp/ModulosPoliciales.json
Modulos policiales (v1GamlpModulosPolicialesJsonGet)
Up 
get v1/gamlp/Salud.json
(v1GamlpSaludJsonGet)
